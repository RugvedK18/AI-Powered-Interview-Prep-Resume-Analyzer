## 🌟 Core Features

### 🔐 Secure Authentication
*   **JWT-Based Security:** Protected API endpoints and robust user session handling using JSON Web Tokens.
*   **User Profiles:** Secure registration and login workflows to isolate personal data and uploaded resumes.

### 📄 Intelligent Resume Parsing & Processing
*   **File Upload Integration:** Seamless handling of document uploads directly through the frontend interface.
*   **Automated Content Extraction:** Background processing to read and structures unstructured resume text accurately.

### 🧠 Gemini AI Insights & Gap Analysis
*   **Job Description Matching:** Compares your resume directly with real-world job roles to measure alignment.
*   **Skill Gap Detection:** Instantly identifies missing keywords, framework proficiencies, or technical concepts.
*   **ATS Optimization Scores:** Rates your resume compatibility based on standard Applicant Tracking System parameters.

### 🤖 AI Interview Simulator
*   **Targeted Question Generation:** Dynamically builds situational, behavioral, and deeply technical questions using Gemini AI.
*   **Custom Persona Prep:** Tailors interview questions to specifically match the target role seniorities or industries.

### 🖨️ PDF Generation
*   **Puppeteer Export Engine:** Generates clean, production-grade, ATS-optimized PDF layouts of your newly updated resumes.
